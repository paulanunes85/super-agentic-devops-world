## What you will learn

- Chart your evolution path as a developer
- Master the terminal with Bash and PowerShell
- Create and manage containers with Docker and Docker Compose
- Write unit, integration, and end-to-end tests
- Understand Open Source licenses and how to contribute
- Work with SQL and NoSQL databases
- Apply best practices for versioning and documentation
- Understand how the internet works (TCP/IP, HTTP, DNS)
- Explore the main programming languages and their use cases
- Choose frameworks and libraries for your projects
- Manage packages and dependencies with npm, pip, and NuGet

- Worlds 1 and 2 complete
