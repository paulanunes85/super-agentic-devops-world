## What you will learn

- Set up and use VS Code as your development environment
- Control your code versions with Git (init, add, commit, branch, merge)
- Collaborate on projects in GitHub with repositories, Issues, and Pull Requests
- Automate builds and deploys with GitHub Actions and CI/CD
- Explore Azure and its main cloud services
- Understand the role of AI in development with Azure AI and GitHub Copilot
- See how all tools connect in the complete development flow

## Prerequisites

- None! This is the starting point for absolute beginners
- Just a computer with internet access
