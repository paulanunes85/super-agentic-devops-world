## What you will learn

- Differentiate between development, staging, and production environments
- Understand how APIs work and HTTP methods (GET, POST, PUT, DELETE)
- Protect applications with secrets, HTTPS, SSL/TLS, and Azure Key Vault
- Comprehend DNS, domains, and name resolution
- Apply agile methodologies (Scrum, Kanban) in your daily workflow
- Understand DevOps culture and Infrastructure as Code (Terraform, Bicep)
- Monitor applications with logs, metrics, and Azure Monitor
