## O que voce vai aprender

- Configurar e usar o VS Code como ambiente de desenvolvimento
- Controlar versoes do seu codigo com Git (init, add, commit, branch, merge)
- Colaborar em projetos no GitHub com repositorios, Issues e Pull Requests
- Automatizar builds e deploys com GitHub Actions e CI/CD
- Conhecer o Azure e seus principais servicos de nuvem
- Entender o papel da IA no desenvolvimento com Azure AI e GitHub Copilot
- Visualizar como todas as ferramentas se conectam no fluxo completo

## Pre-requisitos

- Nenhum! Este e o ponto de partida para iniciantes absolutos
- Apenas um computador com acesso a internet
