## O que voce vai aprender

- Diferenciar ambientes de desenvolvimento, staging e producao
- Entender como APIs funcionam e os metodos HTTP (GET, POST, PUT, DELETE)
- Proteger aplicacoes com secrets, HTTPS, SSL/TLS e Azure Key Vault
- Compreender DNS, dominios e resolucao de nomes
- Aplicar metodologias ageis (Scrum, Kanban) no dia a dia
- Entender a cultura DevOps e Infrastructure as Code (Terraform, Bicep)
- Monitorar aplicacoes com logs, metricas e Azure Monitor

## Pre-requisitos

- Mundo 1 completo (VS Code, Git, GitHub, Actions, Azure)
