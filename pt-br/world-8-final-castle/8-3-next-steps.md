## Prologo: A Cena Pos-Creditos

Os creditos rolaram. A musica de vitoria tocou. A Princesa foi salva. Sofia olhou para a tela com um sorriso satisfeito.

Mas entao -- como em todo bom jogo -- uma cena pos-creditos apareceu.

A tela escureceu e uma mensagem surgiu:

```
+--------------------------------------------------+
|                                                  |
|         "Thank you, Sofia!"                      |
|                                                  |
|    Voce completou o Mushroom Kingdom.            |
|    Mas o universo dos jogos e infinito.           |
|                                                  |
|    Novos mundos aguardam:                        |
|    - O Mundo da Certificacao                     |
|    - O Mundo da Comunidade                       |
|    - O Mundo da Especializacao                   |
|    - O Mundo da Lideranca                        |
|                                                  |
|    A aventura nunca termina de verdade.           |
|    Ela apenas muda de fase.                      |
|                                                  |
|              [PRESS START]                       |
|                                                  |
+--------------------------------------------------+
```

*"Voce salvou a Princesa, Sofia. Mas como todo heroi sabe -- depois do 'The End' sempre vem um 'New Game+'. As habilidades que voce aprendeu sao a BASE. Agora e hora de especializar, certificar, conectar e liderar."*

## 1. A Arvore de Habilidades se Estende

### 1.1 O Mapa Alem do Mushroom Kingdom

O Mushroom Kingdom (os 8 Worlds deste guia) cobre os fundamentos e conceitos intermediarios. Mas alem dele existem reinos inteiros a explorar:

```
MAPA DOS REINOS ALEM DO MUSHROOM KINGDOM:

  [Mushroom Kingdom]     <-- Voce esta aqui (completou!)
         |
    +----+----+----+
    |    |    |    |
    v    v    v    v
  [DK   [Star [Bean [Dark
  Isle]  Rd]   Bean] Moon]

  DK Isle    = Especializacao Tecnica Profunda
  Star Road  = Certificacoes e Reconhecimento
  Bean Bean  = Comunidade e Networking
  Dark Moon  = Lideranca e Mentoria
```

### 1.2 Os 4 Caminhos de Evolucao

Apos completar os 8 Worlds, voce pode seguir 4 caminhos. Eles nao sao exclusivos -- voce pode (e deve) combinar:

| Caminho | Descricao | Analogia Mario | Ideal Para |
|---------|-----------|---------------|-----------|
| Especialista Tecnico | Aprofundar em uma area (backend, frontend, IA, DevOps) | Dominar TODAS as estrelas de um unico World | Quem ama resolver problemas complexos |
| Certificado | Obter certificacoes reconhecidas pelo mercado | Coletar todas as medalhas do jogo | Quem quer validacao profissional |
| Comunitario | Contribuir para open source, participar de eventos, ensinar | Jogar multiplayer e ajudar outros jogadores | Quem ama colaborar e compartilhar |
| Lider Tecnico | Liderar times, tomar decisoes de arquitetura, mentorar | Ser o Player 1 que guia o time | Quem quer impactar organizacoes |
