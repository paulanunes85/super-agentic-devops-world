## O que voce vai aprender

- Tracar seu caminho de evolucao como desenvolvedor
- Dominar o terminal com Bash e PowerShell
- Criar e gerenciar containers com Docker e Docker Compose
- Escrever testes unitarios, de integracao e end-to-end
- Entender licencas Open Source e como contribuir
- Trabalhar com bancos de dados SQL e NoSQL
- Aplicar boas praticas de versionamento e documentacao
- Compreender como a internet funciona (TCP/IP, HTTP, DNS)
- Conhecer as principais linguagens de programacao e seus usos
- Escolher frameworks e bibliotecas para seus projetos
- Gerenciar pacotes e dependencias com npm, pip e NuGet

## Pre-requisitos

- Mundos 1 e 2 completos
